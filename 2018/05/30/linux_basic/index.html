2. linux简介

2.1 Linux概述

Linux是一个通用操作系统。一个操作系统要负责任务调度、内存分配、处理外围设备I/O等操作。操作系统通常由内核和系统程序（设备驱动、底层库、shell、服务程序等）两部分组成。

Linux内核是芬兰人Linus Torvalds开发的，于1991年9月发布。而Linux操作系统作为Internet时代的产物，它是由全世界许多开发者共同合作开发的，是一个自由的操作系统（注意是自由不是免费）。

2.2 Linux 优点

  • 通用操作系统，不跟特定的硬件绑定。
  • 用C语言编写，有可移植性，有内核编程接口。
  • 支持多用户和多任务，支持安全的分层文件系统。
  • 大量的实用程序，完善的网络功能以及强大的支持文档。
  • 可靠的安全性和良好的稳定性，对开发者更友好。

2.3 Linux 发展史

  • Ken Thompson / Dennis Ritchie —> Unix (1970s)
  • Xerox（施乐） —> GUI / 鼠标 / 以太网
  • 1980s — IBM PC
  • Unix —> BSD —> AIX (IBM)
  • Unix —> System V —> Solaris (SUN)
  • Uinx —> System III —> HP-UX (HP)
  • Unix —> BSD —> FreeBSD —> Macintosh (Apple) —> macOS —> iOS / watchOS / tvOS
  • Unix —> Minix (Andrew Tanenbaum) —> Linux (Linus Torvalds) —> Android (Google)

2.4 linux 发型版本

  • Redhat

  • CentOS — 服务器 — 远程访问

2.5 操作系统

  • 个人操作系统 - Windows / macOS
  • 服务器操作系统 - 商用Unix / Linux / Windows Server

3. Linux基础命令

Linux系统的命令通常都是如下所示的格式：

例如：

  • who / w / last / who am i 查看最近登录
  • 查看自己使用的Shell - ps

Shell也被称为“壳”，它是用户与内核交流的翻译官，简单的说就是人与计算机交互的接口。目前很多Linux系统默认的Shell都是bash（Bourne Again SHell），因为它可以使用Tab键进行命令补全、可以保存历史命令、可以方便的配置环境变量以及执行批处理操作等。

  • 查看命令的说明 - whatis
  • 查看命令的位置 - which / whereis。
  • 查看帮助文档 - man / info / apropos
  • adduser / passwd / userdel 创建用户、修改密码、删除用户
  • su 用户名 切换用户(switch user)
  • logout / exit 退出登录
  • date / cal 查看日期日历
  • write / wall / mesg 发消息 （ctrl + d 结束会话）
  • clear 清除屏幕
  • history 查看历史命令
    • ! 命令编号 再执行该命令\键盘上下箭头
    • history -c 清除历史命令
  • reboot 重启系统(init 6)
  • shutdown 关机 (init 0)

4. linux文件和文件夹操作

  • pwd 打印工作目录（print working directory）
  • cd 切换目录（change directory ）
    • 注意相对路径和绝对路径
  • ls 列出目录下的内容( list dictionary )
    • -l 长格式查看
    • -a 查看所有文件（隐藏文件）
      • 在linux中以点开头的文件和文件夹是隐藏的
    • -R 递归式查看 遇到文件夹要递归的列出文件夹下的内容
    • -n
  • cat - 查看文件内容（连接多个文件）concatenate
  • touch 创建空文件或者修改已有文件的最后访问时间
  • mkdir 创建文件夹（make dictionary）
    • -p 当没有父文件时要创建子文件
  • rm 删除(remove)
    • -f 强制删除
    • -r 递归删除（删文件夹需要制定该操作）
  • rmdir 删除空文件夹（remove empty directory）
  • wget 通过网络获取文件
    • wget https://www.python.org/ftp/python/3.7.3/Python-3.7.3.tar.xz
    • wget https://www.python.org/ftp/python/3.7.3/Python-3.7.3.tgz
  • xz 文件压缩、解压缩（xz）
    • z 压缩
    • d 解压缩
      • 例如：xz -d Python-3.7.3.xz
  • tar 文件归档/解归档
    • xvf 解归档(将一个文件拆成多个文件)
      • 例如：tar -xvf Python-3.7.3.tar
    • cvf - 创建归档文件（将多个文件合并成一个文件）
  • wc 查看文件行数、单词数、字符数(word count)
    • l 查看行数
    • w 查看单词数
    • c 查看字符数
  • sort 文件排序
    • -r 降序（默认是从小到大 - 升序）
  • uniq - 文件去重（相邻相同的内容会被过滤掉）unique
  • head / tail 查看文件的开头/结尾部分
  • more / less - 分页查看文件
    • 例子：cat -n taobao.html | more
  • diff - different - 比较文件的差别
  • | 管道(将多个进程连接起来、把前一个命令的输出作为下一个命令的输入)
  • 一个大于号 输入重定向
  • 两个大于号 追加输出重定向
  • 2> 错误输出重定向
  • 2>> 错误追加重定向
  • cp - copy - 拷贝文件
  • mv - move - 移动文件（同一个路径下移动相当于是重命名操作）
  • 查找命令（文件操作）
    • grep 搜索字符串（支持正则）
    • find 搜索文件
      • name 根据名字搜索
      • size 根据大小搜索
      • atime 根据类型搜索 d/f/l/s
      • mtime 最后修改时间
      • ctime 创建时间
      • atime 最后访问时间
  • chown 改变文件所有者
  • chmod 改变文件可执行权限

5. 目录结构

  1. /bin - 基本命令的二进制文件
  2. /boot - 引导加载程序的静态文件
  3. /dev - 设备文件
  4. /etc - 配置文件
  5. /home - 用户主目录的父目录
  6. /lib - 共享库文件
  7. /lib64 - 共享64位库文件
  8. /lost+found - 存放未链接文件
  9. /media - 自动识别设备的挂载目录
  10. /mnt - 临时挂载文件系统的挂载点
  11. /opt - 可选插件软件包安装位置
  12. /proc - 内核和进程信息
  13. /root - root账户主目录
  14. /run - 存放系统运行时需要的东西
  15. /sbin - 超级用户的二进制文件
  16. /sys - 设备的伪文件系统
  17. /tmp - 临时文件夹
  18. /usr - 用户应用目录
  19. /var - 变量数据目录

6. linux 系统下的软件安装

  • 包管理工具 - yum / rpm / apt
    • yum search nginx - 从默认的仓库搜索有没有指定的软件
    • yum install -y nginx - 安装软件
    • yum remove -y nginx - 卸载软件
    • yum info nginx - 查看软件相关信息
    • yum list installed - 查看已经安装的软件
    • yum update nginx - 更新软件

7. Vim的使用

  1. 启动vim
    • vim 文件名

  2. vim的三种模式：

    • 命令模式 - 按键代表命令
      • —i或者a—>编辑模式
      • —:或者/—> 末行模式
    • 编辑模式- 键入内容 —Esc—> 命令模式
    • 末行模式 - 搜索/配置/执行系统命令 —Enter—> 命令模式
      • set nu - 显示行号 - set nonu
      • syntax on - 显示高亮语法 - syntax off
      • set ts=4 - 修改制表键的空格数
      • set ruler - 显示光标所在的位置 - set noruler
  3. 命令模式下的相关操作
符号 含义
G 光标移到文件末尾 / 500G - 光标移到第500行
gg 光标移到行首
hjkl 左下右上（前面可以加数字）行
Ctrl+y / Ctrl+e 移动一行
Ctrl+f / Ctrl+b - 翻一页
0 光标到行首
$ 光标到行尾
w 光标移到下一个单词
dd 删除光标所在行
100dd 从光标所在行向下删除100行
dw 删除一个单词
d0 从光标所在的位置删除到行首
d$ 从光标所在的位置删除到行尾
yy 复制光标所在行
10yy 从光标所在行开始复制10行
p 粘贴 5p - 粘贴5遍
u 撤销
ctrl+r 恢复

  1. 映射快捷键：(在.vimrc添加)

    • map gg99999dd (删除所有代码)
    • inoremap main if name__ == ‘main‘:

  2. 多文件操作：

    • :ls 查看所有打开的文件
    • :b 数字 指定显示哪个文件
    • :vs 垂直拆分
    • :sp 水平拆分
    • Ctrl + w 两次 切换窗口
    • :qa 退出所有窗口
    • vim -d 文件1 文件2 打开多文件进行版本比较

  3. vim查找（命令模式下）（支持正则）

    • / enter 从上往下搜
      • n是往下 N是往上
    • ? enter 从下往上查找

  4. vim替换（末行模式）

    • 1,$ 第一行到最后一行
    • c confirm
    • g global(一行出现多处匹配全部找出)
    • i ignore case 忽略大小写
      例如：全文替换idx 为index 在确认后替换

  5. 宏操作

  • 命令模式下 按 qa (a是寄存器的名字)开始录制宏
  • 播放宏：
    数字(代表播放几遍) @a(a是寄存器的名字)
